介绍

Compose是用于定义和运行多容器 Docker应用工具

通过Compose 可以使用YML文件来配置应用程序所需要的所有服务

使用步骤

1.需要有一个镜像

  • dockerhub 拉镜像
  • dockerfile 自己制作镜像

2.docker-compose定义构成应用程序的服务

  • 写一个yaml文件，来启动多个组件（容器）

3.启动docker-compose
